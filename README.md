This package is intended to be used with `sprint-digital/boilerplate-api-ap`
## Installation

You can install the package via composer:

```bash
composer require sprint-digital/boilerplate-crud-generator
```

## Usage

```bash
php artisan make:crud post
```

## Testing

```bash
composer test
```
